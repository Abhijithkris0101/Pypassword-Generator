# Pypassword-Generator
A secure password generator — born in Python, lives in the browser.

----------------------------------------------------------------------------------------------------------
Started as a simple Python terminal script built while learning the basics. Evolved into a full interactive web app with a clean 
dark UI, typewriter animation, and real-time strength indicator.

The core logic — character pools, random selection, and Fisher-Yates shuffling — stays true to the original Python code, 
just rebuilt in JavaScript for the browser.

Built this to learn how backend logic translates to frontend, and ended up going deep into what actually makes passwords 
secure — entropy, pseudorandomness, cryptographic hashing, and why the secrets module beats random every time.

A small project. A lot learned.

-------------------------------------------------------------------------------------------------------------------
